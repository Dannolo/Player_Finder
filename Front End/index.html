HOW IT WORKS ?

Player finder is an application build to let you find your favourite FG players' matches.

Search the player you need

Write in the bar your player's nickname and wait for the results.

Get for events their matches

If you can't find an event you can add it and it will be stored in our Database.

Find a video for that specified match

We search, if possible, the match for you on Youtube and, if we get a match, we will redirect you.
